Network Protocols

Protocols are rules that help devices communicate. One way to think about it is how humans use language to communicate. Sometimes people speak a different language that you can't understand. Protocols are the common language that computers use to understand one another.

TCP/IP

This is the main language of the internet! It splits data into packets and sends them to the right address.

HTTP and HTTPS

These are the rules for web browsing. HTTPS is like HTTP with extra security, like a lock on your door.

FTP (File Transfer Protocol)

FTP is like a mail carrier for files, moving files from one computer to another.
