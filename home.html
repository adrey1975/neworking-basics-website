Welcome to Networking Basics

Learn the essentials of computer networking in a few easy steps.

Why this course?

The goal of this short course is to introduce a complete beginner to the "invisible connections" that help computers talk to each other.


What's a network?

A network is a collection of computers, servers, mainframes, network devices, and other devices connected to one another to allow the sharing of data. Think of a network like a big web that connects different devices so they can share information. Something that may help you visualize how a network functions is to think of computers like a house, you send a mail (data) using the postal system (network)! In the next few sections we'll go over the different Network Types, that type of Devices that connect to a network, what protocols are, and some troubleshooting methods!
